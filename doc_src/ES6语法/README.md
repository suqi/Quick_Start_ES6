# ES6语法
ES6的基本语法和老版本风格相当不同,越来越像python,c#等高级语言,并且加入了生成器等新工具.
使用ES6可以大大提高开发效率,遗憾的是各个平台对ES6的支持不完全,但即便如此,利用babel也可以使用全部ES6特性,
高效的编程.
